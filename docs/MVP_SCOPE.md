# MVP Scope for Jirani App

## Overview
Jirani is a community dispute resolution platform that facilitates mediation between neighbors and community members.

## Core Features
- User registration and profiles
- Dispute submission
- Mediation scheduling
- Agreement tracking
- Offline-first functionality

## MVP Deliverables
- Basic Android app with dispute submission
- Simple mediation interface
- Local data storage
- Basic UI components

## Out of Scope for MVP
- Advanced analytics
- Multi-language support beyond initial
- Integration with external services
